Monazm (منظم) - Subscription Manager
Monazm is a streamlined Dart "Flutter"-based platform designed to help users track and manage their various digital subscriptions in one place.
Key Features
• Subscription Tracking: Easily add subscriptions (e.g., YouTube, Netflix) with custom names.
• Smart Billing Cycles: Supports monthly,yearly,etc billing cycles to automatically track renewal dates.
• Automated Calculation: Calculates the expiry/next billing date based on the starting date provided.
• Discovery Page: Dedicated section for subscription recommendations and new service suggestions.
Tech Stack
• Language: Dart
• Core Logic: Date handling and subscription management.
How to Use Monazm:
1. Enter the name of your subscription.
2. Select the billing frequency. 
3. The app will display when your next payment is due.
